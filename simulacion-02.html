Lineas de campo

El campo electrostatico es invisible para el ojo humano, pero eso no signiica que sea impercesptible para la mente humana, pero entonces ¿Como representar este fenomeno?. La resputesta a la que se llego fueron las lineas de campo, que operan como:

  • Cargas iguales se repelen.
  • Cargas diferentes se atren.
